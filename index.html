Smart Blinds Automation Device

Arduino-based automated blinds system using light sensing, motor control, and 3D-printed components.

Project Overview

This project is an automated blinds device that reads ambient light levels using a photoresistor and opens or closes blinds using a motor-driven spool mechanism attached to 3D-printed parts.

Skills Used

  • Arduino programming
  • CAD design
  • 3D printing
  • Mechanical design
  • Basic electronics
  • Prototype testing

Design Features

  • Photoresistor-based light detection
  • Motorized opening and closing system
  • 3D-printed rod/spool components
  • Physical integration with window blinds
